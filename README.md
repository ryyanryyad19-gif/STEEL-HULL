# نموذج هيكل قاطرة بحرية · STEEL HULL

موقع مشروع تخرج شعبة هندسة بناء وإصلاح السفن — المعهد الفني للمنشآت البحرية ببورسعيد.

## الملفات
- `index.html` — كل الموقع (الكود + المحتوى + النموذج 3D مدمج)
- `assets/img/` — صور المشروع
- `assets/model.stl` — نسخة احتياطية من النموذج
- `.nojekyll` — يخبر GitHub Pages ألا يعالج الموقع

## للتعديل
عدّل `index.html` مباشرة، احفظ، ارفع — ينعكس على الموقع.
